# Problem 1:   Multiples of 3 and 5


## Examples

Default run: "Find the sum of all the multiples of 3 or 5 below 1000."


```
$ ./prog
233168
```

Sum of multiples of 3 or 5 below given argument:

```
$ ./prog 360
30060
```


```
$ ./prog 345
27600
```
